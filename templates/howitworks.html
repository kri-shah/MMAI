About

Welcome to the UFC Fight Prediction project! This initiative leverages data science and machine learning to predict the outcomes of UFC fights. By analyzing historical fight data and fighter statistics, we aim to provide accurate predictions on the probability of a fighter winning a match.

Data Collection

The data for this project is meticulously gathered from various online sources, including UFC statistics and fight records. Here's an overview of our data collection process:

Fighter Statistics:

  • Metrics: Height, age, reach, stance, significant strikes landed per minute, striking accuracy, takedown average, takedown accuracy, takedown defense, submission average, and fight record.
  • Excluded Metric: Weight/weight. Fighters must be in the same weight class to be sanctioned to fight, so the model doesn't consider their weight. Also by excluding weight, users can create fun fantasy matchups, in which the model predicts who will win purely based on the skillsets of two fighters.
  • Method: Data is collected using web scraping techniques, primarily with BeautifulSoup and requests libraries.

Fight Outcomes:

  • Data: Historical fight data, including fight outcomes (win, loss, draw), fight method (KO/TKO, submission, decision), and fight-specific statistics.

Data Preprocessing

Once collected, the data undergoes several preprocessing steps to ensure it is suitable for machine learning:

Data Cleaning:

  • Process: Remove fighters with missing values in key statistics and convert dates of birth to fighter ages.
  • Consistency: Clean up data formats for uniformity.

Feature Engineering:

  • Categorical Variables: Convert variables such as stance and fight method to numerical representations using one-hot encoding.
  • Normalization: Scale numerical features to a consistent range using MinMaxScaler.

Data Merging:

  • Comprehensive Dataset: Merge fighter statistics with fight outcome data to create a robust dataset for model training.

Predictive Model

Our predictive model is built using TensorFlow, a powerful deep learning framework. The model architecture and training process involve:

Model Architecture

  • Neural Network: The neural network is designed with multiple layers to capture the complex relationships between fighter statistics and fight outcomes.
  • The architecture includes an input layer, three hidden layers each with 73 neurons using ReLU activation, and an output layer with a single neuron using sigmoid activation to predict the probability of a fighter winning.

Training and Validation

  • Output: The model outputs the probability of a fighter winning a match based on input features such as fight statistics and fighter attributes.
  • Clarity: Predictions are scaled and rounded for clarity. The model achieved an accuracy of 69.41% on the test data, making this a significant metric indicating its predictive capability.

Prediction

  • Output: The model provides the probability of a fighter winning a match based on input features.
  • Clarity: Predictions are scaled and rounded for clarity.
